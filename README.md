## Sobre o Repositório

Este espaço foi criado para documentar a construção de aplicações web completas (Single Page Applications + APIs RESTful). O foco é estabelecer uma comunicação eficiente entre cliente e servidor, explorando boas práticas de arquitetura, injeção de dependências e roteamento.

> O repositório está logicamente dividido em dois ecossistemas: a API desenvolvida em **Java** e a interface de usuário construída em **TypeScript**.

## Tecnologias e Arquitetura

**Camada de Back-end (`/backend`)**
- Criação de APIs REST com **Spring Web**
- Persistência de dados com **Spring Data JPA** e Hibernate
- Arquitetura baseada em camadas (Controllers, Services, Repositories, DTOs)

**Camada de Front-end (`/frontend`)**
- Arquitetura baseada em Componentes do **Angular**
- Consumo de API utilizando `HttpClient`
- Rotas dinâmicas e *Data Binding* (One-way e Two-way)
